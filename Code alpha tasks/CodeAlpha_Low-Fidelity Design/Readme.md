# Task 1: Low-Fidelity Wireframe Design

## Overview
This project focuses on creating low-fidelity wireframes for an internship and job portal mobile application. The wireframes were designed to establish the application's structure, layout, navigation flow, and user experience before moving to visual design.

## Objective
The objective of this task was to create a clear and functional blueprint of the application by defining screen layouts and user interactions.

## Screens Included
- Login Screen
- Home Screen
- Internship List Screen
- Search Screen
- Profile Screen
- Certificate Screen

## Design Process
1. User Flow Planning
2. Information Architecture
3. Wireframe Creation
4. Layout Testing
5. Navigation Validation

## Tools Used
- Figma

## Key Features
- Simple and intuitive navigation
- User-friendly layout structure
- Clear content hierarchy
- Consistent screen flow
- Mobile-first design approach

## Outcome
The low-fidelity wireframes provided a solid foundation for the application's visual design and prototype development.
