# Task 2: High-Fidelity UI Design

## Overview
This project involves designing a high-fidelity user interface for an internship and job portal mobile application. The design transforms wireframes into visually appealing screens with proper typography, colors, icons, and interactive elements.

## Objective
The objective of this task was to create a modern and user-friendly interface that enhances usability and provides an engaging user experience.

## Screens Included
- Login Screen
- Home Screen
- Internship List Screen
- Search Screen
- Profile Screen
- Certificate Screen

## Design Elements
- Modern UI Components
- Consistent Color Palette
- Professional Typography
- Interactive Buttons
- Responsive Layout Design
- Clean Visual Hierarchy

## Tools Used
- Figma

## Features
- Easy Navigation
- Internship Discovery
- Search Functionality
- User Profile Management
- Certificate Access
- Clean and Professional Interface

## Design Principles Applied
- Consistency
- Accessibility
- Simplicity
- User-Centered Design
- Visual Hierarchy

## Outcome
The high-fidelity design successfully converted the wireframes into a polished user interface ready for prototyping and user testing.
